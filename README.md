# Novoterskaya

Промо-сайт бренда природной минеральной воды Novoterskaya — премиальная светлая история «Баланс четырёх стихий» (Воздух, Лёд, Земля, Огонь).

**Живая версия:** https://StepanCEO.github.io/Novoterskiy/

## Стек

Статичный сайт без сборки: `index.html`, `css/style.css`, `js/main.js`. Шрифты — Google Fonts (Cormorant Garamond, Manrope, IBM Plex Mono).

## Локальный запуск

Любой статичный сервер, например:

```bash
npx serve -l 4321 .
```

Затем открыть http://localhost:4321

## Возможности

- 8 экранов по ТЗ: hero, четыре стихии, путь воды, минеральный состав, история происхождения, доверие и качество, коллекция, финальный бренд-экран.
- Переключение языков RU / EN без перезагрузки страницы.
- Фирменная «водяная нить» — SVG-линия, заполняющаяся по мере прокрутки.
- Адаптив (desktop / laptop / tablet / mobile), режим reduced-motion, cookie-баннер, SEO-метатеги и микроразметка.
